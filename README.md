# Vote plugin
Waterfall上で動く、投票プラグインです

サポートしているバージョン: **1.19~**

---
## 仕様
プレイヤーは一人一票の投票ができ、あとから別のプレイヤーに投票が可能。(ただし以前投票したプレイヤーへの投票は無くなる)  
システムタイムでの0時0分になると投票がリセットされます  

## コマンド
- /vote <プレイヤー名> - プレイヤーに投票します。
- /votetop - 現在の投票ランキングを表示します。
